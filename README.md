# Knowledge Sharing Platform – Frontend

## 📌 Overview
This is the frontend for a Knowledge Sharing Platform that allows users to create and explore technical articles enhanced using AI.

The platform enables:
- Article creation
- AI-assisted writing
- AI-generated summaries
- Search & discovery

---

## ⚙️ Tech Stack
- ReactJS
- Axios
- React Router
- Rich Text Editor

---

## 🧠 Application Flow

Users can:
1. Register & Login
2. Create Articles using rich editor
3. Improve content via AI
4. View summaries generated by AI
5. Edit/Delete their own articles
6. Explore articles via search

Frontend communicates with backend REST APIs.
